## Cel projektu
Głównym celem jest stworzenie profesjonalnej wizytówki online, która spełnia wysokie standardy UX/UI oraz techniczne (semantyka, responsywność, czysty kod). Projekt jest gotowy do wykorzystania w procesach rekrutacyjnych.

## Technologie i narzędzia
Projekt został zbudowany przy użyciu nowoczesnych standardów webowych:
* **HTML5**: Zastosowanie semantycznych tagów (`<header>`, `<section>`, `<article>`, `<nav>`, `<footer>`) dla lepszej struktury i SEO.
* **CSS3**:
    * Układy oparte na **Flexbox** oraz **CSS Grid**.
    * Podejście **Mobile-First** (responsywność dostosowana do smartfonów, tabletów i monitorów).
    * Zastosowanie zmiennych CSS dla łatwej konfiguracji kolorystyki.
    * Płynne animacje przejść oraz efekty najechania (:hover).
* **Vanilla JavaScript (ES6+)**:
    * Obsługa mobilnego menu (hamburger).
    * Płynne przewijanie do sekcji (Smooth Scroll).
    * Dynamiczne podświetlanie aktywnej sekcji w menu (ScrollSpy).
    * Filtrowanie projektów w czasie rzeczywistym.
    * Animacje pojawiania się elementów przy przewijaniu (Intersection Observer API).
* **Zasoby zewnętrzne**:
    * Google Fonts (czcionka Inter).
    * Font Awesome (ikony).
